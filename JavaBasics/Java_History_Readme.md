Java was developed by Sun Microsystems by James Gosling, released in 1995, and became popular due to its platform independence using JVM and bytecode.
It is Initially named as Oak and later renamed to Java.

WORA - Write Once Read Anywhere
Java code is compiled to Byte code.
Same program runs on any OS.

Java Editions:
Java SE - Core Java
Java EE(Jakarta EE ) - Enterprise Application
Java ME - Mobile & Embedded Systems

Java Versions:
Java 1.0 (1995) -  First Version
Java 5 - Generics, Enums
Java  8 - Lambdas, Streams
Java 11 & 17 - Long Term Support Versions
